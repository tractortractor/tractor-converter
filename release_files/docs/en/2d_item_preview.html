How to convert videos and images to Vangers item preview *.avi

  1. Download and install latest version of GIMP and FFmpeg.
  2. Open modes/2d_item_preview/to_preview_avi.bat file in any text editor.
    • Replace value of PATH_TO_INPUT_VIDEOS with path to folder with videos and images to convert.
    • Replace value of PATH_TO_GIMP with path to gimp-console executable.
    • Replace value of PATH_TO_FFMPEG with path to ffmpeg executable.
    On Windows some characters in path, "!" for example, will not be treated properly by batch script, so only letters and digits should be used. *.bat scripts are replaced by *.sh scripts in Linux and macOS versions.
  3. Make sure that all source videos and images in PATH_TO_INPUT_VIDEOS folder have 340x255 resolution. Only formats supported by ffmpeg must be used for videos and images.
  4. Launch modes/2d_item_preview/to_preview_avi.bat.
  5. Now modes/2d_item_preview/out_avi contains converted *.avi. You can replace original *.avi in data/resource/video folder of Vangers. You should backup original data before doing so. See docs/common/2d_item_preview folder for tables of *.avi usage.
