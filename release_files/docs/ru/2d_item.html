Как редактировать *.bmp изображения предметов

  1. Откройте modes/path_to_vangers_data_dir.bat в любом текстовом редакторе. Замените значение переменной PATH_TO_VANGERS_DATA_DIR на путь к папке "data" Вангеров. Она содержит файл "game.lst", папку "resource" и другие ресурсы игры. В Windows некоторые символы в пути, "!" например, будут неправильно восприниматься batch скриптом, поэтому желательно использовать только буквы и цифры. *.bat скрипты заменены *.sh скриптами в Linux и macOS версиях.
  2. Запустите modes/2d_item/bmp_to_tga.bat.
  3. Теперь папка modes/2d_item/intermediate_tga содержит *.tga изображения предметов, и их можно редактировать. Смотрите docs/common/2d_item/items.ods. Если хотите заменить одно из изображений, смотрите ниже.
  4. Запустите modes/2d_item/tga_to_bmp.bat.
  5. Теперь папка modes/2d_item/out_bmp содержит отредактированные *.bmp изображения. Вы можете скопировать содержимое этой папки в папку "data" Вангеров, чтобы заменить оригинальные изображения на отредактированные. Стоит создать резервную копию оригинальных данных перед тем, как это делать.

Важно

  • RLE сжатие должно быть отключено при сохранении *.tga изображений.
  • Все *.tga изображения должны быть 8-битными и должны использовать цветовую палитру default_resources/2d_item/items_outside.pal. У изображений созданных с modes/2d_item/bmp_to_tga.bat уже есть эта палитра.
  • Цвет с индексом 0 в палитре является прозрачным цветом.
  • Если непрозрачные пиксели изображения окажутся за пределами слота предмета в инвентаре, это приведет к графическим багам.
  • Если разрешение изображения больше 800x600, курсор может "застрять" после выбора предмета в игре.
  • Если хотите заменить оригинальное изображение на новое, измените размер или обрежьте новое изображение, чтобы оно подходило под слот предмета, а затем сконвертируйте в 8-битный *.tga, используя палитру одного из оригинальных *.tga.
